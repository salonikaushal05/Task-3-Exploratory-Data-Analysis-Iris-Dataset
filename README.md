# Task 3: Exploratory Data Analysis (EDA) on Iris Dataset

## Objective

The objective of this task is to perform Exploratory Data Analysis (EDA) on the Iris Dataset to understand data patterns, feature relationships, and species characteristics using statistical analysis and visualizations.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Dataset

The Iris Dataset is a well-known dataset used for data analysis and machine learning. It contains 150 observations of iris flowers belonging to three species:

* Setosa
* Versicolor
* Virginica

### Features

* Sepal Length (cm)
* Sepal Width (cm)
* Petal Length (cm)
* Petal Width (cm)
* Species

## EDA Steps Performed

1. Loaded the Iris Dataset.
2. Examined dataset shape and structure.
3. Generated summary statistics.
4. Checked for missing values.
5. Analyzed species distribution.
6. Created visualizations including:

   * Count Plot
   * Pair Plot
   * Correlation Heatmap
   * Histograms
   * Box Plots
   * Scatter Plot
7. Identified patterns and relationships among features.
8. Generated key insights from the dataset.

## Key Findings

* The dataset contains 150 observations and 5 columns.
* No missing values were found in the dataset.
* Each iris species contains 50 samples.
* Petal Length and Petal Width have a strong positive correlation.
* Setosa species is clearly distinguishable from the other species.
* Virginica generally has larger petal dimensions compared to the other species.
* Correlation analysis revealed strong relationships among petal-related features.

Exploratory Data Analysis was successfully performed on the Iris Dataset.
